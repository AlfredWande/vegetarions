analyse module

class analyse.Analysis(data)[source]

Bases: object

Classe pour effectuer diverses analyses statistiques sur un DataFrame de recettes.

data

Le DataFrame contenant les données des recettes.

Type:

pd.DataFrame

calculate_correlation_matrix()[source]

Calcule et affiche la matrice de corrélation pour les variables numériques du DataFrame.

calculate_interaction_means()[source]

Calcule les moyennes des interactions steps-ingredients pour les recettes végétariennes et véganes.

high_calorie_analysis()[source]

Analyse les recettes très caloriques en comparant les résumés nutritionnels des recettes végétariennes et non-végétariennes, ainsi que pour les recettes véganes et non-véganes.

pairwise_correlations()[source]

Calcule et affiche les corrélations de Pearson entre les paires de variables nutritionnelles.

point_biserial_correlation()[source]

Calcule et affiche les corrélations point-bisériales entre une variable binaire (végétarienne ou végane) et des variables nutritionnelles.

t_test_calories()[source]

Effectue des tests t de Student pour comparer les moyennes des calories entre les recettes végétariennes et non-végétariennes, ainsi que pour les recettes véganes et non-véganes.

t_test_interactions()[source]

Effectue un test t de Student pour comparer les interactions entre les étapes et les ingrédients entre les recettes végétariennes et non-végétariennes, ainsi que pour les recettes véganes et non-véganes.

t_tests_high_calories()[source]

Effectue des tests t de Student pour comparer les variables nutritionnelles entre les recettes végétariennes et non-végétariennes, ainsi que pour les recettes véganes et non-véganes parmi les recettes très caloriques.
